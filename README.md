# Fly_Bird 项目

## 项目简介
Fly_Bird 是一个基于 Python 开发的鸟类识别系统，旨在通过图像处理和机器学习技术，实现对鸟类的自动识别和分类。

## 仓库结构
```
Fly_Bird/
├── 答辩PPT/                  # 项目答辩演示文稿
│   └── 项目答辩ppt.pptx
├── 项目APP/                  # 项目应用程序
│   ├── FlyBird安装程序.zip    # 应用程序安装包
│   └── FlyBird运行视频.mp4    # 应用程序运行演示视频
├── 项目工程/                 # 项目工程文件
│   ├── 测试脚本/              # 自动化测试脚本
│   └── flyBrid.7z            # 项目源代码压缩包
└── 项目文档/                 # 项目相关文档
    ├── 胡瑛杭-项目组测试工作总结报告.doc
    ├── 胡瑛杭-项目组软件使用说明书.doc

    └── 胡瑛杭-项目项目工作总结报告.doc
```

## 功能特点
- 基于图像处理的鸟类识别
- 支持多种鸟类品种识别
- 友好的用户界面
- 高性能的识别算法
- 完整的测试脚本

## 如何使用

### 安装应用程序
1. 下载 `项目APP/FlyBird安装程序.zip` 文件
2. 解压文件
3. 运行安装程序，按照提示完成安装
4. 启动应用程序即可使用

### 查看运行视频
- 观看 `项目APP/FlyBird运行视频.mp4` 了解应用程序的使用方法

### 查看项目文档
- `项目文档/胡瑛杭-项目组软件使用说明书.doc` - 详细的软件使用说明
- `项目文档/胡瑛杭-项目组需求分析报告.doc` - 项目需求分析
- `项目文档/胡瑛杭-项目组面向对象分析报告.doc` - 面向对象分析
- `项目文档/胡瑛杭-项目组面向对象设计报告.doc` - 面向对象设计
- `项目文档/胡瑛杭-项目组测试工作总结报告.doc` - 测试工作总结
- `项目文档/胡瑛杭-项目项目工作总结报告.doc` - 项目工作总结

### 查看源代码
1. 下载 `项目工程/flyBrid.7z` 文件
2. 解压文件，即可查看项目源代码

### 运行测试脚本
1. 进入 `项目工程/测试脚本/` 目录
2. 运行相应的测试脚本，例如：
   ```bash
   python test_black_box.py
   python test_boundary.py
   python test_compatibility.py
   python test_performance_monitoring.py
   python test_response_time.py
   ```
3. 或者运行 Sikuli 测试脚本：
   ```bash
   # 使用 Sikuli IDE 打开 .sikuli 文件并运行
   ```
